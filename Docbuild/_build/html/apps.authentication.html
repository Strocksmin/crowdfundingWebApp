Класс AuthConfig наследуется от AppConfig из фреймворка Django и используется для конфигурации приложения apps.auth.

Функция принимает объект запроса request и идентификатор пользователя id. Она выполняет сброс пароля пользователя

apps.authentication.views.ResetPasswordLink(request, uidb64, token)[исходный код]¶

Функция выполняет декодирование uidb64 в строку, затем получает пользователя user с помощью этого значения uid. Если пользователь существует и токен является действительным, то идентификатор пользователя устанавливается в сеансе и происходит перенаправление на страницу сброса пароля passwordReset. Если пользователь не существует или токен недействительный, возвращается HttpResponse с сообщением «Ссылка на сброс пароля недействительна!».

Параметры:

  • request – объект HttpRequest, содержащий информацию о запросе

  • uidb64 – строка, содержащая закодированный идентификатор пользователя

  • token – строка, содержащая токен для проверки действительности ссылки на сброс пароля

Результат:

Если пользователь существует и токен является действительным,

то идентификатор пользователя устанавливается в сеансе и происходит перенаправление на страницу сброса пароля passwordReset. Если пользователь не существует или токен недействительный, возвращается HttpResponse с сообщением «Ссылка на сброс пароля недействительна!».

apps.authentication.views.deleteAccount(request)[исходный код]¶

Функция удаляет учетную запись пользователя. :param request: объект HttpRequest, содержащий информацию о запросе :return: Если пользователь не авторизован, функция перенаправляет на страницу входа.

Если метод запроса POST и форма действительна, функция удаляет учетную запись пользователя,

удаляет идентификатор пользователя из сеанса и перенаправляет на страницу входа.

Если метод запроса не POST, функция возвращает форму удаления учетной записи пользователя и рендерит шаблон

Delete_account.html.

Если метод запроса POST и форма недействительна, функция возвращает форму удаления учетной записи пользователя,

сообщение об ошибке и рендерит шаблон Delete_account.html.

apps.authentication.views.emailPasswordReset(request)[исходный код]¶

Функция отправляет на email пользователя ссылку для сброса пароля. :param request: объект HttpRequest, содержащий информацию о запросе :return: Если запрос отправлен методом POST и форма валидна, то возвращается HttpResponse с сообщением о необходимости подтверждения email.

Если запрос отправлен методом POST и форма не валидна, то возвращается HttpResponse с сообщением об ошибке. Если запрос отправлен методом GET, то возвращается HTML-страница с формой ResetPasswordEmailForm.
