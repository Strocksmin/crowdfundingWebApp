Базовые классы: Model

Класс Project_Report представляет отчет о проекте, который был сделан пользователем. Класс наследуется от models.Model.

Атрибуты:¶

report - отчет о проекте, выбирается из списка возможных значений, хранится в виде строки (CharField). project - внешний ключ на проект (ForeignKey). Указывает на Project, на который создан отчет.
